# CodeStart - Landing Page

Projeto simples de landing page desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Cabeçalho com menu de navegação.
- Seção principal com slogan e botão de chamada para ação.
- Área sobre o curso.
- Formulário de contato.
- Consulta de endereço utilizando a API ViaCEP.
- Mensagem de confirmação enviada com JavaScript.
- Layout responsivo.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- API ViaCEP
